# S126 HIT137 Assignment 03 - Sydney Group 27
**Pictomatchy**

A spot-the-difference game built with Python and tkinter for HIT137 Assignment 03 (Sydney Group 27).

## Requirements

- Python 3.12.4
- See [requirements.txt](requirements.txt) for dependencies

## Setup & Run

```bash
python -m venv venv
.\venv\Scripts\activate  # Windows
source venv/bin/activate  # macOS/Linux
pip install -r requirements.txt
python main.py
```
